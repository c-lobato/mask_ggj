# Masks of Taldea

Jogo desenvolvido na Global Game Jam que aconteceu na Escola Politécnica - UPE.
Consiste em um jogo de plataforma em que o player coleta máscaras que o levam para outra fase, com diferentes poderes e jogabilidade em cada fase.

## Detalhes Técnicos

Desenvolvido na Game Engine Godot (v4.6) com C# para scripting.
